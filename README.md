# minimax_OT
Code for paper "A Swiss Army Knife for Minimax Optimal Transport", ICML'20

 # Dependencies:
* Numpy >= 1.18.1
* POT >= 0.6.0
* CVXPY >= 1.0.25
* MOSEK >= 9.1.9
* Umap >= 0.1.1
* Scikit-Image >= 0.16.2
* Scikit-Learn >= 0.22.1
*  [Subspace Robust Wasserstein](https://github.com/francoispierrepaty/SubspaceRobustWasserstein) code
